# Solar Performance Analysis (Real Plant Data)

This repository contains real-time solar plant performance analysis based on actual site data.

## Key Analysis Covered:
- PR (Performance Ratio) calculation
- CUF (Capacity Utilization Factor)
- DC & AC loss analysis
- Grid unavailability impact
- Inverter vs Plant generation comparison
- MFM (Main Meter) data validation
- Daily & Monthly performance tracking

## Tools & Skills Used:
- Microsoft Excel (Advanced)
- Pivot Tables & Charts
- SCADA Data Analysis
- Performance Monitoring & Reporting

## Key Highlights:
- Identified generation losses and performance gaps
- Analyzed inverter-level and plant-level data
- Created structured KPI dashboards
- Supported O&M decision-making with data insights

## Objective:
To improve solar plant efficiency by analyzing real operational data and identifying actionable improvements.
